## 这软件到底能干什么

先说核心功能，用大白话讲：

**把大文件变小，但看起来还是一样**。你发一份 50MB 的 PPT，它压完可能只剩 5MB，打开来图片还是清晰的，字体排版还在，啥都没少。这是它的核心卖点。

具体支持的格式：PDF、Word（.doc/.docx）、PowerPoint（.ppt/.pptx）、Excel（.xls/.xlsx）、JPEG、PNG、TIFF，Windows 版还额外支持直接压缩 ZIP 包里的文件，不用先解压。

**批量处理**是它另一个让人上头的功能。单次最多可以处理将近一万个文件，一次把整个项目文件夹扔进去，喝口茶的时间就搞定了。做汇报、整理归档、给客户发文件，这功能真的省命。

**右键菜单集成**也挺贴心的——不用打开软件，直接在资源管理器里右键点文件，一键压缩，没有任何多余步骤。

Windows 版还有一个特别功能：**Outlook / Lotus Notes 邮件附件自动瘦身**。发邮件时附件会自动被压缩后再发出去，收件方完全无感知，你也不用手动操作，它默默帮你搞定。

Mac 版深度整合了 Finder，同样支持右键直接压缩，文件隐私方面也很让人放心——所有压缩都**离线本地处理**，文件绝对不会上传到任何服务器。
